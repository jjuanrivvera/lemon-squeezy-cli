## lsqueezy alias set

Define an alias (expansion may be multiple words)

```
lsqueezy alias set <name> <expansion> [flags]
```

### Examples

```
  lsqueezy alias set ords 'orders list'
```

### Options

```
  -h, --help   help for set
```
